# Packs & Lists — Preview → Confirm → Commit (WIP)

## Quickstart
1. `npm install`
2. `npm run prisma:gen`
3. `npm run prisma:migrate`
4. `npm run db:seed`
5. `npm run dev` → http://localhost:3000

- Login (dev-only), go to **Lists**, click **Open Pack** → you'll see a *preview* with flip animation.
- Click **Confirm** to commit to your collection (server writes), or **Cancel** to discard.
